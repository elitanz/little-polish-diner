# The Little Polish Diner — concept website

A single-page concept site for **The Little Polish Diner**, a 22-seat family Polish
diner in **Parma, Ohio's** historic Polish Village — built from public information
as a design proposal.

**Live:** https://elitanz.github.io/little-polish-diner/

## About the restaurant
- **Cuisine:** Polish comfort food — six kinds of pierogi pan-fried with onions,
  golabki, kielbasa & kraut, borscht, breaded pork chops, and lake perch every
  Friday. Homemade since 2006.
- **Address:** 5772 Ridge Rd, Parma, OH 44129
- **Phone:** (440) 842-8212
- **Hours:** Tue–Fri 11a–6p · Sat 11a–5p · Sun & Mon closed
- **Notes:** #3 of 74 restaurants in Parma; a Cleveland Magazine pierogi pick.
  Facebook-only web presence.

## Tech
- Single static `index.html` — no build step. Plain HTML/CSS/JS.
- Live Three.js 3D hero (cursor-stirred pierogi field, PBR + image-based lighting)
  that **runs on mobile too** (touch-stir, perf-tuned), plus pinned scroll-telling,
  parallax gallery, and count-up stats — all disabled under `prefers-reduced-motion`.
- A signature Polish folk-floral band as the cultural motif, with a
  Zilla Slab / Mulish / Pacifico type system.
- Full mobile nav (hamburger dropdown).

## Note
Independent concept/demo. Food photography is illustrative (via Unsplash). Name,
address, phone, hours, and the quote are drawn from public listings
(Cleveland Magazine / Tripadvisor) and should be confirmed with the business
before any production use.
